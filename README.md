# Henry Beiker — Portfolio Website

Single-file portfolio: `index.html`. No build step, no dependencies, just open in a browser.

**Design**: Instrument Serif italic hero name, Bricolage Grotesque headings, DM Mono body.
Palette: neutral off-white `#f5f4f0`, deep teal accent `#0e6e59`, teal-dark sections.
